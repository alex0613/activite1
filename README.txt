Pour cette activité, j'ai pris pour exemple la création d'un cv sous forme d'une page html. Je me suis basé sur les exercices du cours sur le html et le css.
Tout d'abord, je me suis connecté sur mon compte Github et j'ai créé un repository du nom de "activite1".
Ensuite, sous j'ai fait un "git clone" afin de télécharger le repository sur mon ordinateur.
J'ai fait plusieurs commits à des moments charnières de la conception: Ajout de code html, ajout de fichiers, ajout du css...
Une fois le projet terminé, je l'ai envoyé sur Github avec la commande "git push origin master".
